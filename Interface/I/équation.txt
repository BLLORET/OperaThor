						=> sqrt() = racine carrée

RESOLUTION:

C'est une équation de degré: 2 
Pour résoudre une équation de degré 2 voici les étapes:
1. On calcule le Delta (note D) :
			D = b² - 4 * a * c 
			avec a = -3, b = -5, c = 0 
Ce qui donne :
			D = -5² - 4*-3*0 = 25
2. On regarde maintenant la valeur de D pour savoir le nombre de solutions que possède l'équation: 
Le D est positif, l'équation a donc deux solutions :
3. Les solutions s'écrivent donc: 
			x1 = (-b + sqrt(D))/ 2*a; x2 = (-b - sqrt(D))/ 2*a 
Ainsi on otient: 
			x1 = (-5 + sqrt(25))/ 2*-3 = -1,67
 			x2 = (-5 - sqrt(25))/ 2*-3 = -0,00
RESULTATS:
 			=> D = 25 
			=> x1 = -1,67 
			=> x2 = -0,00 

4. Nous pouvons maintenant trouver le polynôme factorisé de cette équation:
RAPPEL : f(x) = a * (x - x1) * (x - x2) ou x1,x2 sont les  solutions de f(x) = 0 (trouvées dans 3. )

Ainsi ici notre polynôme factorisé s'écrit:
	f(x) = a* x * ( x - x1) car x2 = 0 
	Donc, f(x) = -3 * x * ( x - 1,67)
5. Nous pouvons aussi obtenir la forme canonique de ce trinôme

RAPPEL : la forme canonique de ce trinôme s'écrit:    C(x) = a * [ ( x + ( b / 2 * a) )² - ( D / 4 * a² )] où D est le delta de cette équation. 

				Ici a = -3 , b = -5 , D = 25
	On obient donc :
		=> C(x) = -3 * [ ( x + (-5 / 2 * -3) )² - ( 25 / 4 * -3² )]
