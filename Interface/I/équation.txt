          3X² + 1X + 3 = 1X² + 2X

Nous devons tout d'abord tout passer à gauche du égal afin de faciliter la résolution de l'équation.
Nous obtenons donc l'équation de la forme:

          2X² - 1X + 3 = 0 

Nous pouvons maintenant passer à la résolution de l'équation.

RESOLUTION:

RESOLUTION:

C'est une équation de degré: 2 
Pour résoudre une équation de degré 2 voici les étapes:
1. On calcule le Delta (note D) :
			D = b² - 4 * a * c 
			avec a = 2, b = -1, c = 3 
Ce qui donne :
			D = -1² - 4*2*3 = -23
2. On regarde maintenant la valeur de D pour savoir le nombre de solutions que possède l'équation: 
Comme le D est négatif, et comme nous sommes dans l'ensemble réel il n'a donc pas de solutions, nous sommes actuellement dans l'ensemble des complexes
5. Nous pouvons aussi obtenir la forme canonique de ce trinôme

RAPPEL : la forme canonique de ce trinôme s'écrit:    C(x) = a * [ ( x + ( b / 2 * a) )² - ( D / 4 * a² )] où D est le delta de cette équation. 

				Ici a = 2 , b = -1 , D = -23
	On obient donc :
		=> C(x) = 2 * [ ( x + (-1 / 2 * 2) )² - ( -23 / 4 * 2² )]
			= 2 * [ ( x - 0,25)² + 1,44]
